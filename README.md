# React-Node

# steps to run the project.

1. Git clone
2. To run Front end:
  - Go to ./Front-end/project
  - run "npm i"
  - run "npm start"
  - project will start runnning on http://localhost:3000/

3. To run Back-end:
 - Go to ./Back-end
 - run "npm i"
 - run "npm run dev"
 - server will start running on http://localhost:8000/
